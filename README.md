
## Projeto

Neste projeto de exemplo criamos uma ideia de adicionar localizações favoritas utilizando o leaflet e react leaflet.



## Como executar o projeto

Primeiramente instale as dependências

```bash
  npm install
```

Primeiramente instale as dependências

```bash
  npm run dev
```
